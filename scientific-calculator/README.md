# Scientific Calculator

## Overview

The Scientific Calculator is a Java-based console application designed to perform both basic and advanced mathematical calculations. It supports arithmetic operations, scientific functions, and maintains a history of calculations for future reference.

The project demonstrates the practical use of Java Functional Interfaces, Lambda Expressions, Collections Framework, and mathematical computation techniques.

---

## Features

* Addition, Subtraction, Multiplication, Division
* Modulus Operation
* Power Calculations
* Square Root Function
* Trigonometric Functions (Sine, Cosine)
* Logarithmic Calculations
* Absolute Value Function
* Ceiling and Floor Operations
* Calculation History Tracking
* Interactive Command-Based Interface

---

## Technologies Used

* Java
* Functional Interfaces (Function, BiFunction)
* Lambda Expressions

* Scanner Class

## How to Run

### Compile

```bash
javac ScientificCalculator.java
```

### Run

```bash
java ScientificCalculator
```

---

## Sample Usage

```text
>> 10 + 5
= 15.0

>> sqrt 25
= 5.0

>> sin 30
= 0.5

>> history
10.0 + 5.0 = 15.0
sqrt(25.0) = 5.0
sin(30.0) = 0.5
```
